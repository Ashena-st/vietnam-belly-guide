# Vietnam Belly Tour

Petit site statique construit à partir du fichier Excel `belly_tour_VN.xlsx`.

## Mise en ligne avec GitHub Pages

1. Crée un nouveau dépôt GitHub.
2. Ajoute à la racine du dépôt tous les fichiers et dossiers contenus ici.
3. Dans GitHub, ouvre **Settings → Pages**.
4. Dans **Build and deployment**, choisis **Deploy from a branch**.
5. Sélectionne la branche `main` et le dossier `/ (root)`, puis enregistre.
6. GitHub affichera l'adresse du site après le déploiement.

Le site utilise la balise `noindex` et un fichier `robots.txt` afin de demander aux moteurs
de recherche de ne pas le référencer. Cela ne constitue pas une protection par mot de passe :
toute personne disposant du lien peut consulter et partager le site.

## Modifier le contenu

Les pages sont en HTML pur, avec une feuille de style dans `assets/styles.css` et les
interactions de recherche/filtrage dans `assets/app.js`. Aucun outil de compilation n'est nécessaire.

## Photos
